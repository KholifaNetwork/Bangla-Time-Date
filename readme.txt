Displays Bangla Time & Date!


== Description == 
Displays Bangla Time & Date, Hijri Date, Bangla Month Display!

= Usage =
- Install and activate the plugin.
- Navigate to: Settings -> Bangla Time & Date.
- Use widgets or shortcodes.

= Features: =
- Bangla Time & Date.
- Hijri Month.
- Gregorian Month.
- Bangla day.
- Bangla season.

= Speciality =
* Its free!
* Very simple.
* Easy admin panel.
* Built-in functionality.
* Wide-range compatibility.
* Easy to use widgets!
* All time & date output in Bangla!
* No configuration or knowledge required!
* Automatic GMT+6 time and date.

== Installation ==

* Method # 1:
1. Navigate to: Plugins -> Add new -> Search.
2. Search for "Bangla TIme & Date".
3. Install and activate plugin.
4. Go to: Settings -> Bangla Time & Date.

* Method # 2:
1. Download plugin (zip file).
2. Upload and install.
3. Activate plugin.
4. Go to: Settings -> Bangla Time & Date.
